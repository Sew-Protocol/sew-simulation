# Adversarial Simulation Evidence

This directory contains reproducible evidence of failure modes in Ethereum dispute-resolution protocols, generated by running the SEW adversarial simulation suite.

These are **not theoretical vulnerabilities**. Each scenario is a deterministic, reproducible simulation that executes against a live contract state machine, checks 13 protocol invariants at every step, and produces a machine-readable JSON output.

## Reproducing the Evidence

```bash
# Start the Clojure gRPC server
nohup clojure -M:run -- -S --port 7070 > grpc-server.log 2>&1 &
sleep 8

# Run a specific scenario
cd python
python invariant_suite.py --scenario F3 --json ../results/evidence/f3-run.json

# Run all 33 scenarios
python invariant_suite.py --json ../results/evidence/full-suite-run.json
```

Every run is stamped with git SHA, Python version, and UTC timestamp. Output is deterministic — the same commit produces identical results on any machine.
